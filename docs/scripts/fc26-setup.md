# `fc26-setup.ps1`

## Goal

Provide one clean FC 26 setup and troubleshooting path for Legion Go users instead of multiple overlapping scripts.

## Source inspiration

This spec consolidates ideas from `fc26-fix-tool.ps1` and `fc26-resolution-fix.ps1`.

## In scope

- explain recommended resolution and display settings
- provide launcher-specific setup guidance
- suggest overlay and background-app checks
- provide a structured troubleshooting menu
- help users verify whether the issue is display, launcher, or performance related

## Out of scope

- cheat detection
- invasive system-wide tuning unrelated to FC 26
- clean reinstall automation

## Planned troubleshooting sections

### Setup baseline

Resolution, fullscreen mode, scaling, and handheld-friendly settings.

### Launcher checks

Steam or EA App path guidance, launch options, and repair directions.

### Interference checks

Overlay, background-app, and fullscreen-conflict guidance.

### Performance follow-up

Pointers to core Legion Go scripts when the issue is not FC 26-specific.

## Safety notes

This script should mostly guide users and optionally open settings pages. It should avoid bundling too many unrelated fixes into one action.

## Success criteria

- one obvious FC 26 entry point
- fewer overlapping workflows
- easier troubleshooting for common Legion Go display issues
